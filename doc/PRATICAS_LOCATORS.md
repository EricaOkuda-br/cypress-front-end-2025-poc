### Boas Práticas de Nomeação de Elementos para Automação de Testes

- Os tipos dos elementos deve conter abreviação de 3 caracteres tudo minúsculas, seguidos do nome para melhor identificação com as iniciais sempre maiúsculas.

- Ex: Tipo Botão - btnTestAcionar

| Tipo de Locator         | Abreviação |
|-------------------------|------------|
| Botão                   | btn        |
| Input                   | ipt        |
| Label                   | lbl        |
| CheckBox                | chk        |
| ComboBox                | cbo        |
| Opção de ComboBox       | opt        |
| Linha                   | row        |
| Imagem                  | img        |
| Link                    | lnk        |
| RadioButton             | rdo        |
| Tíulo                   | tit        |
| Mensagem                | msg        |
| Modal                   | mdl        |
| Alerta                  | alt        |
| Text                    | txt        |
| Iframe                  | ifr        |
| Menu                    | mnu        |
